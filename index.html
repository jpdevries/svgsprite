Nice

This icon will serve a basic PNG image for <noscript> users and be progessively enhanced to a SVG <use> if the SVG feature detection is passed. The markup for this implementation might not be pretty, but it is fully accessible.

<div class="noscript-enhanced icon" data-xlink="assets/img/icons.svg#icon-thumbs-up" data-src="assets/img/src/png/thumbs-up.png" data-alt="Nice">
  <noscript>
    <!-- if we can't detect for SVG we don't know if we have it. serving PNG to be safe -->
    <img "icon" src="assets/img/src/png/thumbs-up.png" alt="Nice">
  </noscript>
</div>

becomes:

<svg role="img" class="icon">
  <title>Nice</title>
  <use xlink:href="assets/img/icons.svg#icon-thumbs-up"></use>
</svg>

or:

<img class="icon" alt="Nice" src="assets/img/src/png/thumbs-up.png">

So if JavaScript is disabled we serve a PNG. If JavaScript is enabled we test for SVG and progressively enhance accordingly
